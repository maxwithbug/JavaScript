The span element is an inline element,(you need to use disply property) ***Inline Elements:*** Default Behavior: Certain HTML elements, like span, a, strong, em, and img, are inline elements by default. This means they: 1.Flow within the content line, fitting only the content they contain. 2.Respect top and bottom margins, but not left and right margins. 3.Don't respond to width and height properties directly.
